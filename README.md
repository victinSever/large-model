# 项目文档
项目名：MagicCreator
介绍：大模型数据的前端可视化输入输出展示


## 使用

- 依赖下载
    ```bash
    yarn add 或 npm/pnpm install
    ```

- 本地dev启动
    ```bash
    npm/pnpm/yarn test
    ```

- 本地打包
    ```bash
    npm/pnpm/yarn build
    ```
